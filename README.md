## ğŸ¨ Design Philosophy: Neo-Brutalism

This portfolio rejects the "safe" soft-shadow, rounded-corner trends of modern web design. Instead, it embraces:

- **Hard Shadows**: `12px 12px 0px 0px rgba(0,0,0,1)` for depth.
- **Thick Borders**: Consistent `3px` to `6px` black borders.
- **Unapologetic Colors**: Using `#34eaff` (Cyan), `#fb2e01` (Red), and vibrant Yellows.
- **Asymmetry**: Intentional offsets and rotations to create visual tension.
